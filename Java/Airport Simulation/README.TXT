
PROJECT TITLE: Airport Simulation 
PURPOSE OF PROJECT: School project that visual simulates the runways at an airport with incoming and outgoing flights.
VERSION or DATE: 2015 
HOW TO START THIS PROJECT: This can be ran using BlueJ
AUTHORS: Jeffrey VanMeter
USER INSTRUCTIONS: Once running it automatically starts the simulation and the only control is to speed up the time or slow it back down using a slider at the bottom.
